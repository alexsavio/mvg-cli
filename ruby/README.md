# MVG CLI - Ruby
Dependencies:
```
    gem install mechanize
    gem install colorize
```

Run it:
```
    ruby mvg.rb Marienplatz
    ruby mvg.rb "Sendlinger Tor"
```

Alternatively you can install mvg directly from the rubygems.org website
```
    gem install mvg
```
now you can type
```
    mvg Marienplatz
```
in your terminal.

Have fun :)
